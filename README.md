# Gaming Portal

A mobile-responsive gaming portal built with Next.js 16 (App Router) and Tailwind CSS.

## Features

- **Sticky Header**: Navigation bar with logo linking to homepage
- **Footer**: Professional footer with About, Privacy Policy, and Terms of Service links
- **Home Page**: Responsive grid layout displaying all games (2 columns on mobile, 4 on desktop)
- **Game Detail Page**: 
  - Preview mode with game image, title, and description
  - Suggested games section
  - Fullscreen play mode with smooth animations
- **Smooth Animations**: Powered by framer-motion

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components (Header, Footer)
- `data/` - Game data constants
- `public/` - Static assets

## Technologies

- Next.js 16 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
